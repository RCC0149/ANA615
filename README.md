# ANA 615 – Data Mining Techniques

**Author:** Randall C. Crawford

This repository contains coursework completed for **ANA 615: Data Mining Techniques**, a graduate-level course focused on discovering patterns, relationships, and predictive structure in data using modern data mining methods.

The course emphasized **practical data mining workflows**, including data preparation, model building, performance evaluation, and interpretation, with a strong distinction between **explanatory** and **predictive** modeling.

### Topics and techniques covered include:

* Exploratory data analysis and statistical data summaries
* Dimension reduction techniques, including **Principal Component Analysis (PCA)**
* Predictive modeling using **multiple linear regression**
* Model evaluation for numeric and classification outcomes (RMSE, MAE, accuracy)
* Classification performance analysis using **confusion matrices, ROC curves, AUC, lift, and gains charts**
* **k-Nearest Neighbors (KNN)** for classification and prediction
* **Cluster analysis**, including hierarchical clustering and k-means
* **Association rules and market basket analysis** using support, confidence, and lift
* **Collaborative filtering** and recommender system fundamentals
* **Ensemble methods**, including bagging, boosting, and random forests
* Cost-sensitive modeling and decision-oriented evaluation

Assignments required applying these techniques using Python-based data mining tools, interpreting results, and evaluating trade-offs between accuracy, complexity, and interpretability, using Python.

This repository reflects applied competency in **data mining, predictive analytics, and model evaluation**, aligned with the learning objectives of ANA 615.
